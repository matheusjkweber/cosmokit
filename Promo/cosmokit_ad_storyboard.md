# CosmoKit Ad Video

## Skill Usage

- `research-synthesis`: applied with local-source fallback because network access is restricted in this environment.

## Local Sources Used

- `brand.md`
- `brand/features.md`
- `cosmokit-privacy/messages/en.json`

- existing screenshots in `cosmokit-privacy/public/screenshots/`

## Creative Direction

- Format: `1080x1350`
- Language: English
- Style: dark macOS-native, violet glow, premium utility software
- Goal: sell CosmoKit as the control center for serious iOS Simulator workflows

## Narrative

1. Hook
   CosmoKit replaces scattered `xcrun simctl` routines with one native control center.
2. Velocity
   Show how fast common QA tasks become when pushes, deep links and location tools are one click away.
3. Reliability
   Position the QA toolkit as a way to repeat edge-case testing without manual setup.
4. Capture
   Tie screenshots and video capture to QA, docs and App Store marketing output.
5. Upgrade Trigger
   Present Pro features as an escalation path for teams that need proxying, overrides and multi-simulator testing.
6. CTA
   Close on “Free on Mac App Store” with a clear product identity.

1. `FOR IOS DEVELOPERS`

## Risks

- Existing recordings are generic product captures, not feature-isolated demos.
- The export is mute. Music or voice-over can be layered later without changing the edit structure.
- The current version is optimized for paid social and product teasers, not a full-length App Store preview.
