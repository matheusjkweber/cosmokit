Características

Todo lo que necesitas, un panel

Reemplaza decenas de comandos xcrun simctl con una interfaz intuitiva.

Captura de Pantalla

Capturas de alta calidad con marcos de dispositivo, biseles y colores personalizados. Exportaciones perfectas para la App Store.

Notificaciones Push

Envía notificaciones push de prueba al instante. Personaliza payload, badge, sonido y categoría.

Grabación de Video

Graba con captura de audio, indicadores de toque y proporciones personalizadas. Exporta como MP4 o GIF.

Deep Links

Prueba universal links y URL schemes personalizados. Guarda links frecuentes para acceso rápido.

Proxy de Red

Inspecciona, intercepta y simula tráfico HTTP/HTTPS. Depura llamadas API en tiempo real.

Simulación GPS

Simula cualquier ubicación del mundo. Crea rutas para probar funciones basadas en geolocalización.

Gestión de Apps

Arrastra y suelta IPAs, navega contenedores, limpia datos, inspecciona URL schemes, transmite logs en vivo.

Apariencia

Alterna entre modo oscuro/claro, tipo dinámico y configuraciones de accesibilidad al instante.

Barra de Estado

Personaliza hora, batería, operador y señal para capturas de pantalla limpias.

Permisos

Restablece y gestiona permisos de la app — cámara, ubicación, notificaciones — sin reinstalar.

Face ID

Simula registro, coincidencia y fallo para pruebas de autenticación biométrica.

Keychain

Navega y gestiona elementos del keychain almacenados por tu app en el simulador.

Galería

Míralo en acción

Una interfaz macOS nativa diseñada para velocidad y claridad.
