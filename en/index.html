Features

Everything you need, one panel

Replace dozens of xcrun simctl commands with an intuitive interface.

Screenshot Capture

High-quality screenshots with device frames, bezels, and custom colors. Pixel-perfect exports ready for App Store.

Push Notifications

Send test push notifications instantly. Customize payload, badge, sound, and category.

Video Recording

Record with audio capture, touch indicators, and custom aspect ratios. Export as MP4 or GIF.

Deep Links

Test universal links and custom URL schemes. Save frequently used links for quick access.

Network Proxy

Inspect, intercept, and mock HTTP/HTTPS traffic. Debug API calls in real-time.

GPS Simulation

Simulate any location worldwide. Create routes to test geo-based features and navigation.

App Management

Drag & drop IPAs, browse containers, clear data, inspect URL schemes, stream live logs.

Appearance

Toggle dark/light mode, dynamic type, and accessibility settings on the fly.

Status Bar

Customize time, battery, carrier, and signal for clean screenshots.

Permissions

Reset and manage app permissions — camera, location, notifications — without reinstalling.

Face ID

Simulate enrollment, match, and mismatch for biometric authentication testing.

Keychain

Browse and manage keychain items stored by your app in the simulator.

Gallery
